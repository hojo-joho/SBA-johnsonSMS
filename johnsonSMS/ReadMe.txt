SBA - Student Management System

This app contains two models, Course and Student. 
These models have DAO interfaces and Service classes to implement their methods. 
A student is able to log in with valid credentials, see their classes, register for new classes, and log out. 

There is a service for setting up the database connection and another for setting up the data. 
The setUp method for data should only be run once on the first time the app runs. 
If this method runs a second time you will get a database error.
